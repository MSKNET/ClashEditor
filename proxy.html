Clash Editor

Proxy 配置

在这里，您需要填充代理节点配置，注意代理配置第一行必须为 Proxy:。

Clash 支持的代理类型有 ss、vmess、trojan、socks5、http 和 snell。

您可以参考提供的样例配置直接在代码编辑器中直接添加，也可以使用 Clash Editor 提供的「添加向导」。


在「继续」下一步之前，必须先点击「检查」按钮

继续 返回
Proxy:
# shadowsocks
# The supported ciphers(encrypt methods):
#   aes-128-gcm aes-192-gcm aes-256-gcm
#   chacha20-ietf-poly1305 xchacha20-ietf-poly1305
#   aes-128-cfb aes-192-cfb aes-256-cfb
#   aes-128-ctr aes-192-ctr aes-256-ctr
#   rc4-md5 chacha20 chacha20-ietf xchacha20
- name: "ss1"
  type: ss
  server: server
  port: 443
  cipher: chacha20-ietf-poly1305
  password: "password"
  # udp: true

- name: "ss2"
  type: ss
  server: server
  port: 443
  cipher: chacha20-ietf-poly1305
  password: "password"
  plugin: v2ray-plugin
  plugin-opts:
    mode: websocket # no QUIC now
    # tls: true # wss
    # skip-cert-verify: true
    # host: bing.com
    # path: "/"
    # mux: true
    # headers:
    #   custom: value

# vmess
- name: "vmess"
  type: vmess
  server: server
  port: 443
  uuid: uuid
  alterId: 32
  cipher: auto # cipher support auto/aes-128-gcm/chacha20-poly1305/none
  # udp: true
  # tls: true
  # skip-cert-verify: true
  # network: ws
  # ws-path: /path
  # ws-headers:
  #   Host: v2ray.com

# trojan
- name: "trojan"
  type: trojan
  server: server
  port: 443
  password: "password"
  # udp: true
  # sni: example.com # aka server name
  # alpn: h2 # or http/1.1
  # skip-cert-verify: true

# socks5
- name: "socks"
  type: socks5
  server: server
  port: 443
  # username: username
  # password: password
  # tls: true
  # skip-cert-verify: true
  # udp: true

# http
- name: "http"
  type: http
  server: server
  port: 443
  # username: username
  # password: password
  # tls: true # https
  # skip-cert-verify: true

# snell
- name: "snell"
  type: snell
  server: server
  port: 44046
  psk: yourpsk
  # obfs-opts:
    # mode: http # or tls
    # host: bing.com
Shadowsocks 添加向导

「Shadowsocks 添加向导」不支持添加使用 Shadowsocks 插件的节点！
请使用添加向导填写相关信息以后手动在编辑器中添加 plugin 和 plugin-opts 字段，或直接使用编辑器填写全部节点信息！

Vmess 添加向导

「Vmess 添加向导」不支持添加 Websocket 相关设置！
请使用添加向导填写相关信息以后手动在编辑器中添加相关字段，或直接使用编辑器填写全部节点信息！

Trojan 添加向导

「Trojan 添加向导」不支持添加 SNI 与 ALPN 相关设置！
请使用添加向导填写相关信息以后手动在编辑器中添加相关字段，或直接使用编辑器填写全部节点信息！

Clash 托管配置添加向导

您可以使用「Clash 托管配置添加向导」从现有的 Clash 在线配置中获取节点信息。「托管配置添加向导」会通过 cors-anywhere.herokuapp.com 请求 URL，仅支持 YAML 格式的 Clash 托管配置，只有 Proxy 相关信息会被导入。
